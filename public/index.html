Animal Crossing Hourly Radio

Welcome to the Animal Crossing Hourly Radio Project

This project is for a little automatic radio that plays music from the popular series Animal Crossing.

This site is powered by Node js, hosted by Firebase, and built with a little Jquery love. I'm sure there are many improvements and suggestions on how to improve the site. Reach out to me if you'd like to offer some!
